## 📋 About This Project

A modern, responsive website redesign concept featuring:
- Image-first design approach
- Full-screen hero sections
- Responsive layouts (mobile, tablet, desktop)
- Modern CSS3 and vanilla JavaScript
- Professional gym branding aesthetic

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## ▶️ Run Locally

This is a static site (no build step).

### Option A: Simple local server (recommended)

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080` and click around from `index.html`.

### Option B: Open the file directly

You can open `index.html` in your browser, but some relative paths/features may behave differently vs serving over HTTP.
